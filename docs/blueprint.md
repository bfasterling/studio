# **App Name**: DocuChat AI

## Core Features:

- Document Upload: Allow users to upload PDF or text documents for analysis.
- Admin Analysis Instructions: Enable the administrator to provide specific instructions for document analysis. Tool can apply reasoning about the information's context.
- Document Analysis: Analyze uploaded documents based on admin-provided instructions, using AI to extract relevant information.
- AI Chat Agent: Create an AI chat agent that can answer user questions based on the content of the analyzed documents.
- Chat Interface: Provide a user-friendly chat interface for users to interact with the AI chat agent. Includes Firestore integration.
- Response Limiting: Ensure the AI chat agent only responds to questions related to the content of the uploaded documents.
- Firebase Integration: Host and manage the application using Firebase services.

## Style Guidelines:

- Primary color: Deep sky blue (#30A2FF) to inspire conversation.
- Background color: Light blue (#D0E7FF), nearly desaturated, for a clean feel.
- Accent color: Blue-violet (#7B2CBF) for highlights and interactive elements.
- Body and headline font: 'Inter' for a modern, neutral look.
- Use minimalist icons to represent document types and chat actions.
- Clean, centered layout to focus on chat content and document viewing.
- Subtle animations for loading messages and new responses.
